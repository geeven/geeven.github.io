mac本地服务器配置

发表于 2016-06-07   |  

在开发过程中，为了测试方便而又无服务器的情况下，可以利用mac本地配置服务器临时测试。

修改配置文件

cd /etc/apache2/              进入 apache2 所在的文件目录;
ls                          查看当前文件夹下的文件目录.
sudo vim httpd.conf         sudo 以管理员的权限打开. vim
/DocumentRoot               查找DocumentRoot (Web服务器的)根路径
shift + v                     选中这一行
y                              复制这一行
p                              在当前行下面粘贴复制那一行
此处需要复制两句，并将当前文本路径更变
/Options Follow             查找这一行
在 Options Follow 中间插入 Indexes  让服务器以列表的形式呈现内容.

配置完毕之后的样子 Options Indexes FollowSymLinks Multiviews

/php  查找php 所在的那一行.
删除行首的 # 号.
esc  退出编辑模式.
:wq  保存并退出 vim 编辑器.

开启及关闭服务器

  • 启动服务器
    sudo apachectl -k restart or sudo apachectl restart

  • 关闭服务器
    sudo apachectl -k stop or sudo apachectl stop

注：如果配置服务器过程中出现以下提示,说明服务器配置成功:

AH00558: httpd: Could not reliably determine the server's fully qualified domain name, using teacherdeiMac.local. Set the 'ServerName' directive globally to suppress this message

httpd not running, trying to start

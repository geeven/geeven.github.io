mac在终端显示当前目录及color

Posted on 2016-09-09

在mac进行软件相关开发时，有时需要在终端显示文件，这时不同文件有不同的color显示会更加利用查看。以及有时需要查找当前文件目录及子目录的情况，因此我们就引用tree

find命令

当未安装tree时，我们可以通过find命令来查找当彰目录文件
find . -print | sed -e 's;[^/]*/;|____;g;s;____|; |;g'

tree

打开终端，粘贴以下命令脚本用于安装tree

Homebrew 会将套件安装到独立目录，并将文件软链接至 /usr/local 。
Homebrew 的所有文件均会被安装到预定义目录下，所以您无需担心 Homebrew 的安装位置。
`/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"`

`brew install wget`
用brew来安装软件了,安装tree命令
`brew install tree`

mac 配色方案:
打开终端:nano .bash_profile

此时会打开文件.bash_profile,在此文件中增加alias ls='ls -hG'，再根据命令提示保存退出。

tree 的使用:
tree -L 3:查看当前目录下三级子目录。

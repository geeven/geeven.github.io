Android环境变量配置

在mac下Android环境变量设置

  1. 首先，假设你已经下载了Android SDK，解压后安装了adb。记住sdk文件夹路径。我个人的路径为/Users/geeven/Documents/AndorSDKADT/adt-bundle-mac-x86_64/sdk/
  2. 在终端中输入命令，进入用户目录
  3. 然后输入命令:geeven$ touch .bash_profile.该命令的作用是如果不存在.bash_profile文件，则创建该文件
  4. 然后输入命令:geeven$ open -e .bash_profile.该命令的作用是用文本编辑器TextEdit打开.bash_profile文件。如果你是第一次配置环境变量，则该文档应该是空的。

  5. 下面要在该文件中加入下面的代码
    export PATH=${PATH}:/Users/geeven/Documents/AndorSDKADT/android-sdk-macosx/platform-tools
    export PATH=${PATH}:/Users/geeven/Documents/AndorSDKADT/android-sdk-macosx/tools

  6. 保存并退出TextEdit。关闭bash终端。

  7. 测试：

    • 测试adb $ adb version
    • 测试android $ android

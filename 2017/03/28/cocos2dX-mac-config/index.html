Cocos2d-x 配置开发环境

使用mac配置Cocos2d-x开发环境

  1. 下载最新版本的工具

    Android Studio

    Cocos2d-x

    Android SDK

    Android NDK

    Apache Ant

  2. 准备

    1. 安装 Android Studio
    2. 解压Android NDK
    3. 将目前为止得到的4个文件夹(cocos2d/SDK/NDK/Ant)拷贝到Applications(应用程序)文件夹中

  3. 配置环境

    1. 运行 cocos2dx文件夹下的 setup.py，提示输入NDK路径.
      Tip:输入NDK的路径，注意在末尾删除空格，补上一个”/“，否则会不成功：
    2. 输入SDK路径，这里不添加斜杠也没事：
    3. 输入ANT的路径，这里需要输入bin的目录，注意末尾删掉空格，补上一个“/”。成功添加ANT路径后，提示输入命令 source /Users/Jay/.bash_profile 使配置生效.根据提示，输入:source /Users/Jay/.bash_profile。这样环境就配置完成。

  4. 创建一个工程

    • 使用终端，创建一个名为MyGame的工程：

  • 创建新项目相关命令:

  1. Build

    1. iOS-Xcode打开相关项目，非常容易
    2. Android-Android Studio编译项目

    进入目录:cd MyGame;

    并执行以及命令:
    cocos compile -p android -android-studio

    出错：

    上一步输入ANT_ROOT路径没输对，再来一次。

    输入路径/Applications/apache-ant-1.10.1/bin/
    开始执行了,等一会儿:

    再次出错:

    终端执行/Applications/android-sdk-macosx/tools/android,再安装一个SDK，再次执行上面命令，完成.

    再打开Android-studio，选择proj.android-studio运行即可!
